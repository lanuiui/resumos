Resumo: Operações com Vetores

Os vetores são grandezas vetoriais utilizadas para representar informações que possuem magnitude e direção. Nas operações com vetores, podemos realizar soma, subtração, multiplicação por escalar e produto escalar.

Soma de Vetores

A soma de vetores é realizada somando as componentes correspondentes dos vetores envolvidos. Sejam os vetores A e B dados por:

A = (Ax, Ay, Az)

B = (Bx, By, Bz)

A soma A + B resulta no vetor C:

C = (Ax + Bx, Ay + By, Az + Bz)

Subtração de Vetores

A subtração de vetores é similar à soma, mas subtraindo as componentes correspondentes. A diferença entre os vetores A e B resulta no vetor D:

D = (Ax - Bx, Ay - By, Az - Bz)

Multiplicação por Escalar

A multiplicação de um vetor por um escalar envolve multiplicar cada componente do vetor pelo escalar. Seja o vetor A e o escalar k:

A = (Ax, Ay, Az)

k é um escalar

O resultado da multiplicação é o vetor E:

E = (k * Ax, k * Ay, k * Az)

Produto Escalar

O produto escalar, também conhecido como produto interno, é uma operação que resulta em um número real. Sejam os vetores A e B dados por:

A = (Ax, Ay, Az)

B = (Bx, By, Bz)

O produto escalar entre A e B é calculado pela fórmula:

A · B = Ax * Bx + Ay * By + Az * Bz
